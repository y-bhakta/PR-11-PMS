
# PMS (Product Management System)

A lightweight Node.js admin panel for managing products, categories and users using server-side EJS views.

- **Stack:** Node.js, Express, EJS, Cloudinary (image uploads), Bootstrap, DataTables
- **Features:** CRUD for products & categories, user auth, image upload, server-rendered admin UI
- **Structure:** routes, controllers, models, views (EJS), public assets

## Quick Start

1. Install dependencies: `npm install`
2. Configure environment variables (create `.env` with DB and Cloudinary keys)
3. Start the app: `npm start` or `node index.js`

See the project files and views in the repository root and the [views](views/) folder.
